# ArduStack Programming Laungage
### ArduStack is a stack based version of the Arduino C++ Language implemented in Python.

| :------------------: | :---------------: | :------: |

`digitalWrite(pin, HIGH/LOW)`
`tone() & noTone()`
`void loop()`
`digitalRead(pin)`
`analogRead(pin)`
`analogWrite(pin, value)`
`delay()`
`Variable Storing`
`for loop`
